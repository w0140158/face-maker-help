Getting Help →

You can randomize the face by pressing Ctrl + R or clicking the "Randomize Face" button.

This will:

  • Randomly select a Hair Style.
  • Randomly select an Eye Shape.
  • Randomly select a Nose Style.
  • Randomly select a Mouth Style.

If you don’t like the random face, click Randomize Face again!
